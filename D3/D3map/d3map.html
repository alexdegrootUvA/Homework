Countries by Population Density (2015)



Interactivity:


I have tried to keep my interactivity simple but very functional. I have chosen for a tooltip to pop up when hovering the map. The textbox that pops up shows both the name of the country the cursor is currently on and the density. In addition, the country shades to a light gray, so it is clear for the user which country he/she is hovering. Furthermore, the stroke of the country becomes slightly thicker and changes color. I have chosen this effect to ensure the user knows which country he/she is hovering on. Lastly, not all countries are included in my map. If the user hovers over a country that is not in the data, no tooltip shows up.
